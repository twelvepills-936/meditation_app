Meditation App
==============

Это экспериментальное мобильное веб‑приложение для медитаций, собраное на React Native + Expo.

Проект спроектирован и реализован при активном участии ИИ, который **хорошо справился с мобильной спецификой**:

- учёл особенности Expo Go и версий SDK / React Native;
- аккуратно обошёл текущие ограничения React Navigation и React 19;
- настроил работу на реальном устройстве iOS через Expo Go;
- помог адаптировать интерфейс под мобильный формат (жесты, размеры, читабельность на маленьком экране).

Основные экраны:

- экран медитаций со списком сессий и логикой доступа по подписке;
- paywall / экран подписки с тарифами и мок‑покупкой;
- AI‑фича «настрой дня» с генерацией аффирмаций под настроение пользователя.

Запуск:

```bash
npm install
npx expo start
```

Далее можно открыть приложение в Expo Go на iOS или в браузере (`npx expo start --web`).
